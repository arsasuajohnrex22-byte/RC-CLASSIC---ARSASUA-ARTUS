Firebase Setup Instructions

  1. Go to Firebase Console and create a new project
  2. Enable Firestore Database (start in test mode)
  3. Enable Storage (for images, optional)
  4. Register a web app and get your Firebase configuration
  5. Copy the config to firebase-config.js
  6. Click "Upload Sample Data" button above to populate your database
  7. Test each endpoint using the buttons
  8. Your main website at index.html will automatically use Firebase data

Note: If Firebase is not set up, the website will automatically use fallback data.
